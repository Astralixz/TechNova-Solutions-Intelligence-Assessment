# Open Sources

## Data Sources

This vault contains intelligence data on TechNova Solutions gathered from publicly available open sources.

### Open Source Intelligence (OSINT) Methods

#### 1. DNS Reconnaissance
- **Subdomain Enumeration:** Public DNS queries
- **Domain Analysis:** WHOIS lookups and public domain records
- **Infrastructure Mapping:** IP address and ASN identification
- **SSL Certificate Analysis:** Public certificate transparency logs

#### 2. Public Records
- **Company Registrations:** Public business registration databases
- **Domain Registrations:** Public WHOIS records
- **Website Analysis:** Publicly accessible website content

#### 3. Professional Networks
- **LinkedIn Profiles:** Publicly available professional profiles
- **Company Pages:** Public company pages on professional networks
- **Public Profiles:** Information from publicly accessible profiles

#### 4. Infrastructure Analysis
- **Public DNS Records:** DNS enumeration and subdomain discovery
- **IP Address Analysis:** Public IP and ASN information
- **Hosting Provider Information:** Publicly available hosting data
- **Technology Stack Identification:** Public headers, certificates, and responses

#### 5. Public Websites
- **Company Websites:** Publicly accessible website content
- **Public Documentation:** Publicly available documentation and resources
- **Public APIs:** Publicly accessible API endpoints

## Data Types from Open Sources
- Domain and subdomain information
- Public IP addresses and hosting information
- Public email addresses (from websites, public profiles)
- Public LinkedIn profiles
- Public company information
- Infrastructure details from DNS reconnaissance
- Technology stack information from public headers

## Limitations
- Only includes information from publicly available open sources
- Does not include data from closed sources or breach databases
- Some information may be incomplete or require further verification
- Additional personnel may exist but not found in available open source data
